Carousel Component

A horizontal scrolling carousel component for the BLV theme that transforms a <ul> element with the class carousel into an interactive scrollable gallery.

Features

  • Native scrolling: Uses browser's native scroll behavior for best performance
  • Touch-friendly: Supports swipe gestures on mobile and tablet devices
  • Snap scrolling: Items snap into place for precise positioning
  • Hidden scrollbar: Clean appearance while maintaining scroll functionality
  • Optional navigation: Add buttons or dots as needed
  • Accessible: Includes proper ARIA labels for screen readers
  • Responsive: Works on all screen sizes and devices
  • Customizable: Multiple size variants available
  • Smooth animations: Fade effects for items coming into view

JavaScript API

The carousel component provides a JavaScript API for programmatic control:

// Initialize all carousels CarouselComponent.init(); // Scroll to specific index const carousel = document.querySelector('.carousel'); CarouselComponent.scrollToIndex(carousel, 2); // Get current index const index = CarouselComponent.getCurrentIndex(carousel);
