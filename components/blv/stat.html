Design Specifications

Visual design details from the Figma component.

Card Container

  • • Width: 389px
  • • Padding: 24px
  • • Background: rgba(255,255,255,0.75) with backdrop-blur(5px)
  • • Border Radius: 4px
  • • Box Shadow: 0px 2px 16px 0px rgba(102,114,125,0.25)
  • • Inner Shadow: 0px 4px 200px 0px inset rgba(255,255,255,0.01)

Typography

  • • Title: 20px, Sohne Halbfett (600), 24px line-height, --color-text-default
  • • Stat Number: 96px, Sohne Halbfett (600), 96px line-height, --color-text-success
  • • Description: 16px, Sohne Buch (400), 24px line-height, --color-text-default
  • • All text has text-shadow: #ffffff 0px 1px 0px

Layout Structure

  • • Container: flex flex-col
  • • Title section: flex gap-10px items-center justify-center
  • • Stat section: padding: 8px 0, flex items-center justify-center
  • • Description section: flex gap-10px items-center justify-center
  • • Icon size: 96px × 96px

Color Mapping

Figma variables mapped to foundation CSS variables.

Text Colors

  • • Text/Default: #181B25 → var(--color-text-default)
  • • Text/Success: #18543C → var(--color-text-success)

Typography Tokens

  • • XL/BOLD_SHORT: Sohne Halbfett, 20px, 600 weight, 24px line-height
  • • MD/REGULAR: Sohne Buch, 16px, 400 weight, 24px line-height

Usage

Basic HTML structure for the stat component.

  </div>

  </div>

    <p class="stat-card__description-text">visits with healthcare providers</p>
  </div>
</div>
