Original Component Reference

Color mapping from the original BaselineVisitCheckMarkButton:

Unchecked State

  • • Border: border-[#4D576F] → var(--color-gray-700)
  • • Text: text-[#4D576F] → var(--color-gray-700)
  • • Focus Border: focus:border-[#4D576F]
  • • Focus Text: focus:text-[#4D576F]
  • • Active Background: active:bg-[#4D576F]
  • • Active Text: active:text-white

Checked State

  • • Border: border-foundation-primary → var(--color-foundation-primary)
  • • Text: text-foundation-primary → var(--color-foundation-primary)
  • • Icon Fill: fill-white
  • • Active Fill: active:fill-foundation-primary
