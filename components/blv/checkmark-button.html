Usage Guidelines

Best practices for implementing checkmark buttons in your applications.

HTML Structure

Use semantic button elements with proper ARIA attributes for accessibility:

<button class="checkmark-button" data-checked="false">
  <svg class="checkmark-button__icon">...</svg>
  <span class="checkmark-button__text">Label</span>
</button>

JavaScript Integration

Update button state and icon dynamically:

// Toggle checked state
toggleCheckmarkButton(buttonElement);

// Set specific state
setCheckmarkButtonState(buttonElement, true);

Accessibility

  • • Use semantic <button> elements for proper keyboard navigation
  • • Include aria-pressed attribute to indicate selection state
  • • Provide clear visual feedback for hover, focus, and active states
  • • Ensure sufficient color contrast for all states
  • • Use descriptive text labels that clearly indicate the action or choice
  • • Group related options with <fieldset> and <legend>
