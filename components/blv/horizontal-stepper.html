Usage Guidelines

Best practices for implementing horizontal steppers in your applications.

HTML Structure

Use semantic HTML with ordered lists and proper ARIA attributes:

<ol class="horizontal-stepper">
  <li class="horizontal-stepper__step horizontal-stepper__step--first">
    <div class="horizontal-stepper__progress" style="width: 50%;"></div>
  </li>
  ...
</ol>

Progress Updates

Update progress by modifying the width style property:

// Update progress to 75%
progressEl.style.width = '75%';

// Complete step
progressEl.style.width = '100%';

Accessibility

  • • Use semantic <ol> elements to indicate step sequence
  • • Include aria-label or aria-describedby for screen readers
  • • Provide text alternatives for progress information
  • • Ensure sufficient color contrast for progress indicators
  • • Consider adding step labels for complex workflows
