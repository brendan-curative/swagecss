Overview

The Flippy Card component provides an interactive card that flips to reveal additional content on the back. It inherits all styles from the base card component and adds 3D flip animation triggered by hover, focus, or tap/click interactions.

Key Features

  • Inherits all base .card styles and behaviors
  • 3D flip animation with perspective transforms
  • Multiple trigger methods: hover, focus, tap/click
  • Keyboard accessible (Tab to focus, Enter/Space to flip)
  • Touch-friendly with mobile optimization
  • Size variants (small, default, large)
  • Programmatic control via JavaScript API
  • Respects prefers-reduced-motion settings

Accessibility Features

Flippy cards are fully accessible:

  • Keyboard Navigation: Press Tab to focus, Enter or Space to flip
  • Screen Readers: Cards announce their flip state with ARIA labels
  • Reduced Motion: Respects prefers-reduced-motion settings
  • Touch Support: Works seamlessly on mobile and tablet devices

CSS Classes

Available CSS classes for the flippy card component:

  • .flippy-card - Main container (add .card to inherit base card styles)
  • .flippy-card__inner - Inner flip container (required)
  • .flippy-card__front - Front face content
  • .flippy-card__back - Back face content
  • .flippy-card__content - Content wrapper within front/back
  • .flippy-card__title - Title styling
  • .flippy-card__text - Text content styling
  • .flippy-card--small - Small size variant
  • .flippy-card--large - Large size variant
  • .flippy-card--flipped - Programmatically flipped state

JavaScript API

The FlippyCardComponent JavaScript API provides the following methods:

  • FlippyCardComponent.init() - Initialize or re-initialize all flippy cards
  • FlippyCardComponent.flipTo(card, showBack) - Flip a specific card to front or back
  • FlippyCardComponent.flipAll(showBack) - Flip all cards on the page
  • FlippyCardComponent.toggle(card) - Toggle a card's flip state
  • FlippyCardComponent.isFlipped(card) - Check if a card is currently flipped

Example usage:

// Flip a specific card to back
const card = document.querySelector('.flippy-card');
FlippyCardComponent.flipTo(card, true);

// Flip all cards to front
FlippyCardComponent.flipAll(false);

// Toggle a card
FlippyCardComponent.toggle(card);

// Listen for flip events
card.addEventListener('flippyCardFlip', (e) => {
    console.log('Card flipped:', e.detail.isFlipped);
});
