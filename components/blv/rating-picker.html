Styling patterns from the original BaselineVisitRatingPicker:

Layout & Structure

  • • Grid: grid grid-cols-3 gap-[10px]
  • • Cards: flex cursor-pointer flex-col items-center justify-between space-y-1
  • • Border: rounded border border-gray-100 bg-white p-2
  • • Transition: transition-all

Colors by Rating

  • • Positive: text-textColor-success → var(--color-text-success)
  • • Neutral: text-textColor-highlight-default → var(--color-text-highlight-default)
  • • Negative: text-[theme(colors.signalOrange)] → var(--color-primary-signal-orange)

Selection States

  • • Selected: font-black [box-shadow:_0px_2px_16px_0px_rgba(102,114,125,0.25)]
  • • Unselected (when any selected): opacity-50

Usage

Basic HTML structure replicating the original component.

<div class="rating-picker">
  <div class="rating-picker__option" data-value="negative">
    <svg class="rating-picker__icon">...</svg>
    <p class="rating-picker__label">I am confused</p>
  </div>
  ...<!-- other options -->
</div>
