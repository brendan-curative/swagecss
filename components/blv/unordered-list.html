Component Reference

Styling details and differences from Ordered List:

Key Differences

  • • Uses .unordered-list__icon instead of .ordered-list__number
  • • Icon container holds a .heroicon element instead of a number span
  • • Icons are sized at 16px (--spacing-16) within the 24px circle
  • • All other styling (backdrop, shadows, colors) identical to Ordered List

Container Styling

  • • List container: 8px vertical spacing between items
  • • Background: rgba(255, 255, 255, 0.75) with backdrop-blur(5px)
  • • Shadow: 0px 2px 16px 0px rgba(102,114,125,0.25), 0px 4px 200px 0px rgba(255,255,255,0.01) inset

Icon Circle Styling

  • • Container: 24px × 24px circular, white background
  • • Border: 1px solid (color set by data-color attribute)
  • • Icon size: 16px × 16px
  • • Icons inherit color from parent

Color Variants

  • • Blue: data-color="blue" → var(--color-primary-sky-blue)
  • • Orange: data-color="orange" → var(--color-primary-signal-orange)
  • • Content text: var(--color-text-default), bold, 18px

Available Icons

Use any heroicon from foundation/icons.css. Common examples:

  • • heroicon-check - Checkmark
  • • heroicon-star - Star
  • • heroicon-heart - Heart
  • • heroicon-user - User profile
  • • heroicon-shield-check - Shield with check
  • • heroicon-calendar - Calendar
  • • heroicon-clock - Clock
  • • See foundation/icons/ for complete list

Usage

Basic HTML structure with icon bullets.

<ul class="unordered-list">
  <li class="unordered-list__item" data-color="blue">
    <div class="unordered-list__icon">
      <span class="heroicon heroicon-check"></span>
    </div>
    <div class="unordered-list__content">
      <p class="unordered-list__text">Item Text</p>
    </div>
  </li>
</ul>
