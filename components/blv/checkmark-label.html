Original Component Reference

Styling patterns from the original BaselineVisitCheckMarkLabel:

Layout & Structure

  • • Container: flex items-center space-x-1
  • • Icon container: flex size-4 items-center justify-center rounded-full border border-skyBlue bg-white
  • • Icon: size-3 text-skyBlue (CHECK variant)
  • • Text: Label + ": " + font-semibold text-textColor-success "Complete"

Colors from Original

  • • Icon border & color: border-skyBlue, text-skyBlue → var(--color-primary-sky-blue)
  • • Icon background: bg-white → var(--color-white)
  • • Complete text: text-textColor-success → var(--color-text-success)
  • • Complete font: font-semibold → var(--font-weight-semibold)

Sizing

  • • Icon container: size-4 → 16px × 16px
  • • Check icon: size-3 → 12px × 12px
  • • Spacing: space-x-1 → 4px gap

Basic HTML structure replicating the original component.

<div class="checkmark-label">

    <svg>...checkmark path...</svg>
  </div>

  </p>
</div>
