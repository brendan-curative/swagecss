How to Enable Track Views

Add two data attributes to your carousel:

  • data-carousel-track-views - Enables slide view tracking
  • data-carousel-controlled-button="buttonId" - ID of the button to control

The controlled button should have disabled attribute initially:

<div class="carousel" 
     data-carousel-track-views 
     data-carousel-controlled-button="myButton">
    <!-- carousel content -->
</div>

<button id="myButton" class="button button--primary" disabled>
    Continue
</button>

Accessibility Features

The Carousel component includes comprehensive accessibility support.

Keyboard Navigation

  • Left Arrow: Navigate to previous slide
  • Right Arrow: Navigate to next slide
  • Tab: Move focus between interactive elements
  • Enter/Space: Activate focused button or indicator

ARIA Attributes

  • role="region" on carousel container
  • aria-label describing carousel purpose
  • aria-hidden on non-active slides
  • aria-live="polite" on slide counter for announcements
  • aria-current on active indicator

JavaScript API

Control carousel behavior programmatically using the JavaScript API.

API Methods

Initialize Carousel

initializeCarousel();

Navigate to Next Slide

carouselNext('carouselId');

Navigate to Previous Slide

carouselPrev('carouselId');

Go to Specific Slide

carouselGoTo('carouselId', slideIndex);

Access Carousel API

const carousel = document.getElementById('myCarousel');
carousel.carouselAPI.next();     // Go to next slide
carousel.carouselAPI.prev();     // Go to previous slide
carousel.carouselAPI.goTo(2);    // Go to slide index 2
carousel.carouselAPI.getCurrentIndex(); // Get current slide index
carousel.carouselAPI.getTotalSlides();  // Get total slide count
carousel.carouselAPI.getViewedSlides(); // Get array of viewed slide indices
carousel.carouselAPI.areAllSlidesViewed(); // Check if all slides viewed
