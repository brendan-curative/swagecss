Original Component Reference

Styling patterns from the original OrderedList component:

Container Styling

  • • List container: space-y-2 → 8px vertical spacing

Item Layout & Structure

  • • Layout: relative flex w-full items-center gap-8 rounded-[4px] px-4 py-3
  • • Background: bg-white/75 backdrop-blur-[5px]
  • • Shadow: shadow-[0px_2px_16px_0px_rgba(102,114,125,0.25),_0px_4px_200px_0px_rgba(255,255,255,0.01)_inset]

Number Circle Styling

  • • Container: flex size-6 items-center justify-center rounded-full border bg-white
  • • Text shadow: [text-shadow:_0px_1px_0px_#FFF]
  • • Font: font-semibold

Color Variants

  • • Blue: border-skyBlue, text-skyBlue → var(--color-primary-sky-blue)
  • • Orange: border-signalOrange, text-signalOrange → var(--color-primary-signal-orange)
  • • Content text: text-lg font-bold text-textColor-default

Usage

Basic HTML structure replicating the original component.

<ol class="ordered-list">
  <li class="ordered-list__item" data-order="1" data-color="blue">
    <div class="ordered-list__number">
      <span>1</span>
    </div>
    <div class="ordered-list__content">
      <p class="ordered-list__text">Item Text</p>
    </div>
  </li>
</ol>
