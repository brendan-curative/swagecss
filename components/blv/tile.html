The Tile component displays provider or service information with visual status indicators. It supports three states: In Network (green check), Cash Card (orange check), and Not Listed (blue info icon). Tiles can be dismissed via the x-circle icon.

In Network State

Displays providers or services that are in-network, using a green check-circle icon.

Design Specifications

Visual design details from the Figma component.

Tile Container

  • • Max Width: 361px
  • • Padding: 8px 16px
  • • Background: rgba(255,255,255,0.75) with backdrop-blur(5px)
  • • Border Radius: 4px
  • • Box Shadow: 0px 2px 16px 0px rgba(102,114,125,0.25)
  • • Inner Shadow: 0px 4px 200px 0px inset rgba(255,255,255,0.01)
  • • Layout: flex row with 16px gap
  • • Min Height: 54px (auto based on content)

Typography

  • • Title: 16px, Sohne Buch (400), 24px line-height, --color-text-default
  • • Subtitle: 12px, Sohne Buch (400), 14px line-height, --color-gray-600

Icon States

  • • Status Icon: 24px × 24px, positioned on left
  • • In Network: heroicon-check-circle, --color-green-700 (#0E6345)
  • • Cash Card: heroicon-check-circle, --color-primary-signal-orange (#FC5C50)
  • • Not Listed: heroicon-information-circle, --color-blue-700 (#074ADF)
  • • Dismiss Icon: heroicon-x-circle, 24px × 24px, --color-blue-700

Content Layout

  • • Content area uses inline-grid with overlapping text
  • • Grid: grid-template-columns: max-content
  • • Grid: grid-template-rows: max-content
  • • Both title and subtitle use grid-area: 1 / 1 (same cell)
  • • Subtitle has margin-top: 24px to appear above title
  • • Max width: 249px

Color Mapping

Figma variables mapped to foundation CSS variables.

State Colors

  • • Foundation/Success: #0E6345 → var(--color-green-700) (In Network)
  • • Foundation/Accent B: #FC5C50 → var(--color-primary-signal-orange) (Cash Card)
  • • Text/Highlight/Depressed: #0C2F91 → var(--color-blue-700) (Not Listed & Dismiss)

Text Colors

  • • Text/Default: #181B25 → var(--color-text-default)
  • • Text/Secondary: #525B64 → var(--color-gray-600)

Usage

Basic HTML structure for the tile component.

In Network Tile

<div class="tile tile--in-network">
  <span class="tile__icon heroicon heroicon-check-circle"></span>
  <div class="tile__content">
    <p class="tile__subtitle">Primary Care Physician</p>
    <p class="tile__title">Rosemary Hahn, MD</p>
  </div>
  <button class="tile__dismiss" aria-label="Dismiss">
    <span class="tile__dismiss-icon heroicon heroicon-x-circle"></span>
  </button>
</div>
