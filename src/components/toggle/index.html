Toggle Component

`Toggle` component renders an interface for toggling a value between two states, exactly as implemented in the Curative UI design system.

Toggle Sizes

The two toggle sizes available in the Curative UI Toggle component.

Large Toggle (Default Size)
Small Toggle

Toggle States

Toggles in different states showing the color and position changes.

Large - Unchecked State
Large - Checked State
Small - Unchecked State
Small - Checked State
Disabled Toggle

Usage

How to implement the Toggle component with the exact Curative UI structure.

Props Interface

export type ToggleProps = { onRequestChangeValue: (v: boolean) => void; size?: ToggleSize; // LARGE (default) | SMALL srOnlyLabel: string; value: boolean; disabled?: boolean; };

HTML Structure

<button class="toggle toggle--large toggle--unchecked" type="button" aria-pressed="false" > <span class="toggle__sr-only">Large Toggle</span> <span class="toggle__indicator"></span> </button>

Size Classes

/* Large (Default) */ .toggle--large /* w-16 (64px width) */ .toggle__indicator /* h-6 w-6 (24px) */ /* translate-x-8 (32px) when checked */ /* Small */ .toggle--small /* w-8 (32px width) */ .toggle__indicator /* h-3 w-3 (12px) */ /* translate-x-3 (12px) when checked */

Color Mapping

/* Unchecked */ background-color: var(--color-text-subdued); /* bg-textColor-subdued */ /* Checked */ background-color: var(--color-foundation-primary); /* bg-foundation-primary */ /* Indicator */ background-color: var(--color-surface-default); /* bg-surface-default */
