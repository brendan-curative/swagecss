Modal Component

The Modal component renders a pop-up dialog that grabs the user's attention, informing them of something or seeking confirmation from them. It supports different style variants, sizes, and interactive elements.

Usage

How to implement and customize the modal component.

Basic HTML Structure

<!-- Trigger Button --> <button data-modal-trigger="my-modal">Open Modal</button> <!-- Modal --> <div id="my-modal" class="modal modal--hidden" aria-hidden="true"> <div class="modal__container"> <div class="modal__panel modal__panel--default modal__panel--md"> <div class="modal__header"> <h3 class="modal__title modal__title--default">Modal Title</h3> <button data-modal-close class="modal__close-button"> <div class="heroicon heroicon-24 heroicon-x-mark"></div> <span class="modal__sr-only">Close</span> </button> </div> <p class="modal__message modal__message--default"> Your message content here. </p> <div class="modal__button-container"> <button data-modal-primary class="modal__button modal__button--primary"> Continue </button> </div> </div> </div> </div>

JavaScript API

// Open a modal programmatically window.ModalComponent.open('my-modal'); // Close a modal programmatically window.ModalComponent.close('my-modal'); // Set loading state window.ModalComponent.setLoading('my-modal', true); // Update modal content window.ModalComponent.updateMessage('my-modal', 'New message'); window.ModalComponent.updateTitle('my-modal', 'New title');

Event Listeners

const modal = document.getElementById('my-modal'); modal.addEventListener('modalOpen', (e) => { console.log('Modal opened:', e.detail.modalId); }); modal.addEventListener('modalClose', (e) => { console.log('Modal closed:', e.detail.modalId); }); modal.addEventListener('modalPrimaryClick', (e) => { console.log('Primary button clicked:', e.detail); });
