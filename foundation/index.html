Usage Guidelines

Best practices for using the foundation system effectively in your projects.

CSS Variables

Use CSS custom properties for consistent styling in custom components:

.custom-component {
  color: var(--color-foundation-primary);
  padding: var(--spacing-16);
  font-size: var(--font-size-lg);
}

Utility Classes

Prefer utility classes for rapid prototyping and layout:

<div class="flex items-center gap-16 p-24 bg-white border rounded">
  <h2 class="heading-lg text-gray-900">Title</h2>
</div>
